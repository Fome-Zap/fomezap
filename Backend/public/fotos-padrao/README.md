# Fotos Padrão para Produtos

Esta pasta contém imagens padrão que os restaurantes podem usar para seus produtos.

## Estrutura:

- **lanches/** - Hambúrgueres, sanduíches, coxinhas, etc
- **pizzas/** - Diversos sabores de pizza
- **bebidas/** - Refrigerantes, sucos, cervejas, etc
- **sobremesas/** - Bolos, sorvetes, doces
- **pasteis/** - Pastéis de diversos sabores

## Como adicionar fotos:

1. Adicione imagens nas pastas correspondentes
2. Formatos aceitos: JPG, PNG, WEBP
3. Tamanho recomendado: 400x400px (quadrado)
4. Nome sugerido: descritivo, ex: `hamburguer-bacon.jpg`

## Fontes de imagens gratuitas:

- https://unsplash.com (buscar: "food", "burger", "pizza", etc)
- https://pexels.com
- https://pixabay.com
