# 3. Introdução

Desenvolver um software de gerenciamento e agendamento de salas com o objetivo de facilitar e otimizar a questão da disponibilidade e utilização das salas da Fatec Jahu (BRASIL, 2018).

Este documento apresenta a concepção, análise e implementação do projeto "Agenda Fatec", desenvolvido como projeto interdisciplinar (PI) no curso de Tecnologia em Desenvolvimento de Software Multiplataforma do Centro Paula Souza — Fatec Jahu. O projeto visa prover uma ferramenta que permita a alunos, professores e auxiliares de docência consultar disponibilidade, solicitar e administrar agendamentos de salas, com foco em usabilidade, desempenho e conformidade com a LGPD (BRASIL, 2018).

## Contexto e motivação

Muitas instituições acadêmicas dispõem de sistemas internos de agendamento, porém grande parte deles não atende de forma direta às necessidades locais. O "Agenda Fatec" foi pensado para ser uma solução personalizada para a Fatec Jahu, com potencial de expansão para outras unidades. O projeto busca reduzir conflitos de uso de salas e melhorar a experiência de reserva por parte dos usuários.


## Escopo do trabalho

Este PI cobre as etapas de levantamento de requisitos, modelagem, implementação (protótipo funcional), validação e documentação. Alguns itens — como integração com sistemas institucionais legados e implantação em produção — são considerados futuros trabalhos a serem tratados após validação do protótipo.
